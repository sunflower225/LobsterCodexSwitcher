# CJFCodexSwitcher v0.1.0

首个公开版本发布，聚焦两个目标：

1. 让新手用户可以快速安装并使用账号切换工具
2. 让 Agent 可以通过稳定的非交互 CLI 直接消费账号数据和切换结果

## Highlights

- 新增统一账号列表页面，启动后直接进入查看余量页
- 新增 `[当前]` 标记、进度条刷新、按序号切换和自动刷新
- 新增 Agent 友好的 CLI：
  - `--list`
  - `--list --json`
  - `--best`
  - `--switch`
  - `--save-current`
  - `--refresh`
- 新增项目内 Skill，指导用户与 Agent 使用本工具
- 支持更标准的 Python console script 安装路径

## Installation

### Homebrew

```bash
brew install mileson/cjfcodexswitcher/cjfcodexswitcher
```

### pipx

```bash
pipx install git+https://github.com/mileson/CJFCodexSwitcher.git
```

## Verification

本次发布前已完成以下验证：

- 交互页面可正常查看余量、切换账号、自动刷新
- 非交互 CLI 可输出排序结果与 JSON
- 项目内 Skill 结构与内容通过校验
- `pyproject.toml` 打包入口可安装并运行
- 本地敏感目录未纳入版本控制

## How To Use

- 新手用户：直接运行 `codex-switcher`
- Agent：优先使用 `codex-switcher --list --json` 和 `codex-switcher --best --json`
- 需要直接切换：运行 `codex-switcher --switch best`
