## ADDED Requirements

### Requirement: Repository SHALL include release-grade README presentation

The repository SHALL include badges, a product screenshot, and concise install commands in the README so first-time visitors can quickly understand and install the project.

#### Scenario: Render badges and screenshot

- **WHEN** a user opens the repository homepage
- **THEN** the README MUST display release-grade badges and at least one product screenshot

### Requirement: Project SHALL provide release notes for the published version

The project SHALL provide a release notes document and use it for the first public GitHub release.

#### Scenario: Create first release notes

- **WHEN** the first public version is published
- **THEN** a release notes document MUST exist and describe highlights, installation, and usage verification
